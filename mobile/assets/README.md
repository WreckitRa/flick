# Assets Folder

This folder should contain:

- `icon.png` - App icon (1024x1024)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `splash.png` - Splash screen (2048x2048 recommended)
- `favicon.png` - Web favicon (48x48 or larger)

## Placeholder Assets

For MVP-0 development, you can use placeholder images:

1. Create solid color images with the Flick yellow (`#FFD93D`)
2. Add a simple emoji or text in the center
3. Use any image editor or online tool

## Production Assets

For production, create professional assets with:
- Flick branding
- High resolution
- Multiple sizes for different platforms
- Follow Apple and Google design guidelines
